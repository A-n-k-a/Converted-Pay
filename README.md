![Image](https://s2.loli.net/2022/08/20/rmPM7kwxUzQpJhb.png)

## 🚀 特点

- 纯前端实现，一键部署，无需安装，无需数据库
- 免维护，无多余配置，一次修改永久使用
- 代码轻量，数据量小，飞速加载 
- 支付软件扫码智能识别，无需手选，且允许切换支付方式
- 用户可对网站样式进行高度自定义
- 提供徽章生成器和二维码下载功能（网页右上角），方便用户使用
